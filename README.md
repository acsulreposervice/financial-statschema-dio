# financial-statschema-dio

## 📋 Sobre o Projeto
Projeto com foco na criação de um **star schema**
originaria de uma tabela sólida.

O projeto contou com processos de transformação de
dados e abstrações sobre modelagem dimensional. 

### 🔍 Desafios Encontrados
Um dos principais desafios foi na abstração de chaves
unicas para evitar relacionamentos **M:N**. 

Posteriormente, essa modelagem será refinada, analisando
e aplicando as melhores práticas dentro do proposto.

## 🛠️ Tecnologias Utilizada

- **PowerQuery**: para transformações específicas nos dados.
- **PowerBI**: para transformações mais gerais e modelagem.
